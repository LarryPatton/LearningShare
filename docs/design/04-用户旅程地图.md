# 用户旅程地图（User Roadmap）

## 1. 用户角色定义

### 1.1 角色分类
博客网站涉及三种主要用户角色：

| 角色 | 说明 | 权限范围 |
|------|------|---------|
| **访客（Guest）** | 未注册/未登录用户 | 浏览文章、查看评论、搜索内容 |
| **注册用户（User）** | 已注册并登录的普通用户 | 访客权限 + 发表评论、点赞、个人中心 |
| **管理员（Admin）** | 网站管理者 | 全部权限 + 内容审核、用户管理、数据统计 |

---

## 2. 访客（Guest）用户旅程

### 2.1 旅程概览
```
访问首页 → 浏览文章列表 → 阅读文章详情 → 查看评论 → 发现感兴趣内容 → 注册/离开
```

### 2.2 详细旅程地图

#### 阶段 1：首次访问
**触点**：搜索引擎、社交媒体、直接输入网址

**用户行为**：
1. 进入首页，查看网站整体风格
2. 浏览顶部导航栏（首页、分类、标签、关于我）
3. 看到首页推荐文章（置顶、精选、最新）
4. 快速浏览文章标题和摘要

**系统响应**：
- 展示精美的首页设计
- 加载最新和热门文章卡片
- 显示分类导航和标签云
- 响应式布局适配不同设备

**用户情绪**：好奇、观望

**痛点**：
- 不清楚网站内容质量
- 不知道是否值得花时间浏览

**优化策略**：
- 首页展示高质量精选文章
- 清晰的分类导航
- 快速加载速度（SSR 优化）

---

#### 阶段 2：内容浏览
**用户行为**：
1. 点击感兴趣的文章
2. 阅读文章内容（图文混排）
3. 查看文章作者信息
4. 浏览文章评论区
5. 查看相关推荐文章

**系统响应**：
- 渲染完整文章内容（Markdown/HTML）
- 显示代码高亮、图片优化
- 展示作者卡片和评论列表
- 推荐相关文章（同分类/标签）

**用户情绪**：专注、满意/失望

**痛点**：
- 文章加载慢
- 图片未优化导致卡顿
- 代码片段不易复制

**优化策略**：
- 图片懒加载 + CDN 加速
- 代码一键复制功能
- 文章目录导航（长文章）
- 阅读进度条

---

#### 阶段 3：深度探索
**用户行为**：
1. 通过分类/标签发现更多内容
2. 使用搜索功能查找特定主题
3. 查看某个作者的其他文章
4. 想要评论但被提示需要登录

**系统响应**：
- 分类页面展示该分类下所有文章
- 搜索结果页面（高亮关键词）
- 作者主页展示其文章列表
- 评论框显示"登录后评论"提示

**用户情绪**：兴趣增强、萌生注册念头

**关键决策点**：是否注册账号？

**转化策略**：
- 优质内容吸引用户回访
- 评论互动吸引用户注册
- 提供第三方快速登录（GitHub、Google）

---

#### 阶段 4：离开或注册
**场景 A：直接离开**
- 原因：内容不符合预期、暂时无注册需求
- 优化：SEO 优化，提升回访率

**场景 B：注册成为用户**
- 进入注册流程 → 转为"注册用户"旅程

---

## 3. 注册用户（User）旅程

### 3.1 旅程概览
```
注册账号 → 完善资料 → 互动评论 → 点赞收藏 → 个人中心 → 持续活跃/流失
```

### 3.2 详细旅程地图

#### 阶段 1：注册与激活
**用户行为**：
1. 点击"注册"按钮
2. 填写注册信息（用户名、邮箱、密码）
3. 提交注册表单
4. 收到验证邮件（可选）
5. 邮箱验证完成

**系统响应**：
- 显示注册表单（实时验证）
- 发送验证邮件
- 注册成功自动登录
- 跳转到个人资料完善页面

**用户情绪**：期待、略有负担（填表）

**痛点**：
- 注册流程复杂
- 邮箱验证等待时间长

**优化策略**：
- 简化注册表单（仅必填项）
- 支持第三方登录（GitHub、Google）
- 注册成功后给予新手引导

---

#### 阶段 2：首次使用
**用户行为**：
1. 完善个人资料（昵称、头像、简介）
2. 浏览首页文章
3. 发表第一条评论
4. 点赞喜欢的文章
5. 查看个人中心

**系统响应**：
- 引导完善个人资料
- 显示用户昵称和头像
- 评论成功提示
- 个人中心展示活动记录

**用户情绪**：新鲜感、探索欲

**关键成功指标**：
- 首次评论完成率
- 7 日留存率

---

#### 阶段 3：日常使用
**用户行为**：
1. 定期访问网站查看新文章
2. 在感兴趣的文章下评论
3. 回复他人评论
4. 点赞文章和评论
5. 查看个人评论历史
6. 修改个人资料

**系统响应**：
- 首页显示"最新文章"动态
- 评论通知（有人回复我）
- 个人中心统计（评论数、点赞数）
- 浏览历史记录

**用户情绪**：归属感、满意

**痛点**：
- 找不到之前阅读的文章
- 评论被回复没有通知

**优化策略**：
- 添加"我的收藏"功能
- 评论回复邮件/站内通知
- 个性化推荐算法

---

#### 阶段 4：流失或忠诚
**场景 A：用户流失**
- 原因：更新频率低、内容质量下降、缺乏互动
- 挽回策略：邮件推送新文章、个性化推荐

**场景 B：成为忠实用户**
- 特征：高频访问、积极评论、主动分享
- 激励措施：用户等级系统（可选）、优质评论者标识

---

## 4. 管理员（Admin）旅程

### 4.1 旅程概览
```
登录后台 → 查看数据统计 → 创建文章 → 管理评论 → 用户管理 → 系统配置
```

### 4.2 详细旅程地图

#### 阶段 1：登录后台
**用户行为**：
1. 从前台点击"后台管理"
2. 输入管理员账号密码
3. 进入管理后台首页（Dashboard）

**系统响应**：
- 验证管理员权限
- 显示统计概览（文章数、用户数、今日访问量）
- 展示最新评论、热门文章

**用户情绪**：掌控感

---

#### 阶段 2：内容创作
**用户行为**：
1. 点击"新建文章"
2. 填写标题、选择分类和标签
3. 使用富文本编辑器撰写内容
4. 上传封面图片
5. 保存草稿或直接发布
6. 预览文章效果
7. 设置文章属性（置顶、精选）

**系统响应**：
- 提供强大的 Markdown/富文本编辑器
- 自动保存草稿（避免丢失）
- 实时预览功能
- 图片上传并自动压缩
- 发布后自动生成 SEO 友好 URL

**用户情绪**：创作满足感

**痛点**：
- 编辑器功能不足
- 图片上传慢
- 没有草稿自动保存

**优化策略**：
- 集成 TinyMCE/Quill 编辑器
- 图片自动压缩和 CDN 上传
- 每 30 秒自动保存草稿
- 支持快捷键操作

---

#### 阶段 3：内容管理
**用户行为**：
1. 查看文章列表（草稿、已发布、隐藏）
2. 编辑已发布文章
3. 删除不需要的文章
4. 查看文章数据（浏览量、点赞数、评论数）
5. 管理分类和标签

**系统响应**：
- 文章列表支持筛选和搜索
- 批量操作（批量删除、修改分类）
- 展示文章数据统计图表
- 分类和标签树形结构管理

**用户情绪**：效率感

---

#### 阶段 4：用户与评论管理
**用户行为**：
1. 查看用户列表
2. 查看用户详细信息
3. 封禁违规用户
4. 审核评论（通过/拒绝/删除）
5. 回复用户评论

**系统响应**：
- 用户列表支持筛选（按角色、状态）
- 显示用户活跃度数据
- 评论列表标记待审核内容
- 评论一键删除/批量操作

**用户情绪**：责任感、掌控感

---

#### 阶段 5：数据分析
**用户行为**：
1. 查看网站整体数据统计
2. 分析文章阅读趋势
3. 查看热门文章排行
4. 查看用户增长曲线
5. 导出数据报表

**系统响应**：
- Dashboard 展示可视化图表
- 数据按时间维度筛选（今日、本周、本月）
- 支持数据导出（CSV/Excel）

**用户情绪**：成就感、洞察力

## 6. 用户流失风险点与挽回策略

### 6.1 访客流失风险点
| 风险点 | 原因 | 挽回策略 |
|--------|------|---------|
| 首页跳出率高 | 内容不吸引人 | 优化首页布局，精选高质量文章 |
| 文章加载慢 | 性能问题 | CDN 加速、图片优化、SSR 渲染 |
| 找不到感兴趣内容 | 导航不清晰 | 优化分类结构、增强搜索功能 |

### 6.2 注册用户流失风险点
| 风险点 | 原因 | 挽回策略 |
|--------|------|---------|
| 长期无新文章 | 更新频率低 | 制定内容发布计划 |
| 评论无人回复 | 缺乏互动 | 管理员及时回复评论 |
| 功能单一 | 缺少吸引力 | 增加收藏、点赞、通知等功能 |

## 7. 用户旅程优化建议

### 7.1 新手引导
- **访客**：首页 Banner 提示"探索优质技术文章"
- **注册用户**：首次登录后弹窗引导（完善资料、发表评论）
- **管理员**：首次进入后台显示功能介绍视频

### 7.2 个性化推荐
- 基于浏览历史推荐相关文章
- 基于点赞和评论推荐类似内容
- 分类订阅功能（用户关注特定分类）

### 7.3 通知机制
- **评论回复通知**：邮件 + 站内消息
- **新文章推送**：订阅用户邮件通知
- **管理员提醒**：待审核评论、异常登录提醒

### 7.4 社交化功能（可选）
- 文章分享到社交媒体
- 用户关注功能
- 评论点赞排行榜

## 9. 关键指标（KPI）

### 9.1 访客指标
- **页面浏览量（PV）**：总访问次数
- **独立访客数（UV）**：独立访问用户数
- **跳出率**：只浏览一个页面就离开的比例
- **平均停留时间**：用户在网站的平均浏览时长
- **注册转化率**：访客注册成为用户的比例

### 9.2 注册用户指标
- **日活跃用户（DAU）**：每日登录用户数
- **7 日留存率**：注册后 7 天仍活跃的用户比例
- **评论活跃度**：每日评论数 / 活跃用户数
- **用户生命周期价值（LTV）**：用户贡献的长期价值

### 9.3 内容指标
- **文章发布频率**：每周/月发布文章数
- **平均阅读完成率**：滚动到文章底部的用户比例
- **文章分享次数**：社交媒体分享量
- **评论互动率**：有评论的文章比例
