# 前台页面线框图设计

> **说明**：本文档使用 ASCII 艺术和文字描述展示页面线框图，不涉及具体视觉设计（颜色、字体等）。

---

## 1. 通用布局组件

### 1.1 顶部导航栏（Header）

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo] 我的博客          首页  分类  标签  归档  关于  [🔍搜索]  │
│                                                    [登录] [注册]  │
└─────────────────────────────────────────────────────────────────┘
```

**元素说明**：
- Logo + 网站名称（左上角）
- 主导航菜单（居中或左对齐）
- 搜索图标（可展开搜索框）
- 用户菜单（未登录显示"登录/注册"，已登录显示头像和昵称）

**响应式**：
- 移动端：汉堡菜单（☰），点击展开侧边抽屉导航

- **侧边栏**：30% 宽度

**响应式**：
- 移动端：单列布局，侧边栏移至底部或隐藏

**布局说明**：
- **面包屑导航**：首页 > 分类 > 文章标题
- **文章头部**：
  - 封面图（可选）
  - 标题（H1）
  - 元信息（作者、时间、浏览数、评论数）
  - 标签列表
- **主内容区**：
  - 文章内容（Markdown 渲染）
  - 代码高亮
  - 图片优化展示
- **互动区**：
  - 点赞按钮
  - 分享按钮
- **评论区**：
  - 评论表单（需登录）
  - 评论列表（支持回复）
  - 评论分页
- **侧边栏**：
  - 文章目录（长文章）
  - 作者卡片
  - 相关推荐文章

**功能特性**：
- 文章目录自动生成并高亮当前章节
- 代码块一键复制
- 阅读进度条（顶部或侧边）
- 图片点击放大预览

## 3. 响应式设计要点

### 3.1 桌面端（Desktop）
- 宽度：≥1200px
- 布局：侧边栏 + 主内容区（70/30 或 75/25）
- 导航：横向导航栏

### 3.2 平板端（Tablet）
- 宽度：768px - 1199px
- 布局：保持侧边栏，宽度调整为 60/40
- 导航：横向导航栏

### 3.3 移动端（Mobile）
- 宽度：<768px
- 布局：单列布局，侧边栏移至底部或隐藏
- 导航：汉堡菜单（☰），点击展开抽屉式导航
- 文章卡片：堆叠布局

---

## 4. 交互状态说明

### 4.1 按钮状态
- **默认**：常规样式
- **悬停（Hover）**：背景色加深或高亮
- **点击（Active）**：按下效果
- **禁用（Disabled）**：灰色不可点击

### 4.2 链接状态
- **默认**：下划线或颜色标识
- **悬停**：颜色变化或下划线
- **访问过**：颜色变暗（可选）

### 4.3 表单验证
- **正确**：绿色边框 + ✓ 图标
- **错误**：红色边框 + ✗ 图标 + 错误提示文字
- **实时验证**：失焦时验证

## 6. 无障碍（Accessibility）要点

- **语义化 HTML**：使用 `<nav>`, `<article>`, `<aside>` 等标签
- **图片 Alt 文本**：所有图片添加描述性 alt 属性
- **键盘导航**：支持 Tab 键导航和回车键操作
- **对比度**：文字与背景对比度符合 WCAG AA 标准
- **屏幕阅读器**：ARIA 标签支持
