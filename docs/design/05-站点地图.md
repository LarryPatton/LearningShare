# 站点地图（Site Map）

## 1. 网站整体结构

博客网站分为 **前台（Front-end）** 和 **后台管理系统（Admin Panel）** 两个主要部分。

```
博客网站
├── 前台（面向所有用户）
│   ├── 内容浏览模块
│   ├── 用户认证模块
│   └── 用户中心模块
│
└── 后台（仅管理员可访问）
    ├── 内容管理模块
    ├── 用户管理模块
    ├── 数据统计模块
    └── 系统设置模块
```

## 7. SEO 优化建议

| 页面类型 | SEO 优化策略 |
|---------|-------------|
| 首页 | 网站名称 + 副标题 |
| 文章详情 | 文章标题 + 网站名称 |
| 分类页 | 分类名称 + 网站名称 |
| 标签页 | 标签名称 + 网站名称 |
| 搜索页 | 搜索关键词 + 搜索结果 |

**Meta 标签**：
- `description`: 页面描述（150-160 字符）
- `keywords`: 关键词（逗号分隔）
- `og:title`, `og:description`, `og:image`: 社交媒体分享优化
